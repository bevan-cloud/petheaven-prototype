Key differences vs M1
The M2 site is a major visual departure from M1. Page background is #F3F4F8 (clean off-white — not the warm parchment of M1). Heading colour is dark teal #204B4A. Price colour is warm orange #D77929. Body text uses #6B6B69 Dark Charcoal (not deep black). Body font is proxima-nova at 16px (not 13px). The overall feel is clean, minimalist, and nature-forward.

02 — Typography

Typography

M2 uses proxima-nova exclusively — a clean humanist sans-serif — for all text: headings, body copy, UI elements, and buttons. No secondary typeface is used. Body text is 16px (vs 13px on M1) — a significantly more spacious reading experience.

04 — Logo

Logo & Usage

The M2 site uses the same FTN wordmark but on a darker teal header (#204B4A) instead of M1's dark green. The off-white page background (#F3F4F8) requires the dark/inverted logo for any light-surface placement.

M2 context difference
Same logo asset as M1, but the header background shifts from #004738 to #204B4A (dark teal). Clear space and usage rules are identical.
✓ Do
  • Use on dark teal (#204B4A) for M2 header/footer
  • Use dark/inverted version on the off-white (#F3F4F8) page
  • Maintain original proportions at all times
  • Use the same tagline: "Change the way you shop, for Good"
✗ Don't
  • Place white logo directly on the parchment background
  • Use the M1 dark green (#004738) as header — M2 uses #204B4A
  • Stretch, crop, or rotate the logo
  • Recolour in orange (#D77929) or other brand colours
  • Add effects, shadows, or outlines
